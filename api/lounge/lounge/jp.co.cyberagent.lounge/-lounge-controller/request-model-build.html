fun requestModelBuild()
Content copied to clipboard

Calls this to request a model update. The controller will schedule a call to buildModels so that models can be rebuilt for the current data.

Only the latest call of this method will trigger a build (via mapLatest), so you don't need to worry about calling this method multiple times.
