suspend fun LoungeBuildModelScope.listRow(headerData: HeaderData? = null, key: Any? = null, controller: LoungeController, presenter: ListRowPresenter = ListRowModel.DefaultListRowPresenter)

Adds a ListRowModel to this scope. Either headerData or key must be provided to properly set the ListRowModel.key. You need to manually invoke the LoungeController.requestModelBuild of the controller for displaying row items.

Parameters

headerData

if provided, set a HeaderItem with the data to the ListRow.

key

if provided, set it as the ListRowModel.key.

controller

set the ListRowModel.controller.

presenter

the ListRowPresenter for the ListRow.
