DeferredLoungeModel

interface DeferredLoungeModel : LoungeModel
Content copied to clipboard

A LoungeModel whose content can be deferred. This is useful if a LoungeModel need to load content asynchronously. We can use a LoungeControllerInterceptor to control whether to display the model after its content ready or display the model directly.
