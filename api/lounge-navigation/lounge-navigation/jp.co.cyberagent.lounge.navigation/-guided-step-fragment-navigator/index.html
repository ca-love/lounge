GuidedStepFragmentNavigator

class GuidedStepFragmentNavigator(context: Context, fragmentManager: FragmentManager, containerId: Int) : Navigator<GuidedStepFragmentNavigator.Destination>
Content copied to clipboard

Navigator that applies proper UiStyle and transitions for GuidedStepSupportFragment. Every destination using this Navigator must set a valid GuidedStepSupportFragment class name with android:name or Destination.setClassName.

In your navigation graph xml file, you can use the tag <guided-step> to define a GuidedStep destination, just like using <fragment> tag.

You must add this navigator to the NavController.getNavigatorProvider first before set the navigation graph that contains <guided-step> tag.
